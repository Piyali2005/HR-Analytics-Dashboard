📊 HR Analytics Dashboard

🔍 Overview

The HR Analytics Dashboard is a data visualization project designed to analyze employee data and provide insights into workforce trends such as attrition, employee demographics, salary distribution, and job satisfaction.

This dashboard helps organizations make data-driven HR decisions by identifying key patterns and improving employee retention strategies.

🚀 Features

📌 Total Employees Overview

📉 Attrition Count & Attrition Rate

👨‍💼 Active Employees Tracking
🎂 Average Age Analysis
💰 Average Monthly Income by Job Role
🏢 Department-wise Employee & Attrition Analysis
🎓 Education Field Distribution
⭐ Job Satisfaction Ratings
📊 Performance Rating by Department
👥 Age & Gender Distribution

🛠️ Tools & Technologies Used

Power BI (for dashboard creation)
Excel / CSV Dataset (data source)
Data Cleaning & Transformation
Data Visualization Techniques


📊 Key Insights

High attrition observed in certain departments like Sales and R&D
Majority of employees belong to Life Sciences and Medical fields
Most employees fall in the 25–44 age group
Job satisfaction varies significantly across different job roles
